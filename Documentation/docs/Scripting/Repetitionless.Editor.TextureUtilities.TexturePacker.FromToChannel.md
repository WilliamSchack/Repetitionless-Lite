# Editor.TextureUtilities.TexturePacker.FromToChannel

## Description

`Unity Editor Only`

Contains which texture channel will be transfered to which other channel

## Members

| Member | Description |
|--------|-------------|
| From | The channel that will be taken from the original texture |
| To | The channel which will be written to on the packed texture |
| FromToChannel(TextureChannel, TextureChannel) | FromToChannel Constructor |

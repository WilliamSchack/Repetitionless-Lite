# Editor.Constants

## Description

`Unity Editor Only`

Contains the main constants used for the Repetitionless asset

## Variables

| Variable | Description |
|----------|-------------|
| PACKAGE_NAME | The package name |
| PACKAGE_PATH | The package path relative to the project path |
| SAMPLES_PATH | The samples path relative to the project path |
| SAMPLES_PATH_ASSETS | The core sample assets path relative to the project path |
| SAMPLES_PATH_BIRP | The Built-In samples path relative to the project path |
| SAMPLES_PATH_URP | The URP samples path relative to the project path |
| SAMPLES_PATH_HDRP | The HDRP samples path relative to the project path |
| DOCUMENTATION_URL | The online documentation url |
| LOCAL_DOCUMENTATION_PATH | The local documentation path relative to the project path |
| GITHUB_URL | The github issues repo url |
| GITHUB_NEW_ISSUE_URL | Link to submit a new issue in the github repo |
| TEXTURE_DATA_FILE_NAME | The texture data asset file name that will be stored along side a material |
| PROPERTIES_FILE_NAME | The material properties asset file name that will be stored along side a material |
| PROPERTIES_TEXTURE_ASSET_NAME | The properties texture file name that will be stored along side a material |
| ARRAY_ASSIGNED_TEXTURES_ASSET_NAME | The array assigned textures texture file name that will be stored along side a material |
| DEFAULT_VARIATION_TEXTURE_NAME_2K | The default variation texture file name with a resolution of 2048x2048 |
| DEFAULT_VARIATION_TEXTURE_NAME_4K | The default variation texture file name with a resolution of 4096x4096 |
| MATERIALS_PER_LAYER_COUNT | How many materials are used per layer |
| COMPRESSED_MATERIAL_VARIABLES_COUNT | Amount of compressed variables per material |
| COMPRESSED_LAYER_VARIABLES_COUNT | Amount of compressed variables per layer |
| TRIPLANAR_KEYWORD | The triplanar keyword for repetitionless materials |

# Editor.Data.RepetitionlessTextureDataSO.LayerTextureData

## Description

`Unity Editor Only`

Contains the texture data for a repetitionless layer

## Variables

| Variable | Description |
|----------|-------------|
| BaseMaterialTextures | The base material textures |
| FarMaterialTextures | The far material textures |
| BlendMaterialTextures | The blend material textures |
| BlendMaskTexture | The blend mask texture for the layer<br />Stored in an array to make it easier to pass into the texture drawer |

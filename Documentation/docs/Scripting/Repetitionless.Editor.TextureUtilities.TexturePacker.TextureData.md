# Editor.TextureUtilities.TexturePacker.TextureData

## Description

`Unity Editor Only`

Contains the data for a texture

## Members

| Member | Description |
|--------|-------------|
| Texture | The texture |
| Disabled | If the texture will be ignored |
| DataTexture | If this texture is a data texture<br />If enabled it will not be treated as srgb |
| NormalMap | If this texture is a normal map<br />If enabled it will not be treated as srgb and will be packed as a normal |
| FromToChannels | Which channels will be copied to which other channels |

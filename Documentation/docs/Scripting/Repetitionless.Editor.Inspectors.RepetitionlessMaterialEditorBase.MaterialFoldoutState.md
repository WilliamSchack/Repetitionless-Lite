# Editor.Inspectors.RepetitionlessMaterialEditorBase.MaterialFoldoutState

## Description

`Unity Editor Only`

Holds the state for each property foldout in a material section

## Variables

| Variable | Description |
|----------|-------------|
| MainProperties | Foldout state for the main properties |
| NoiseProperties | Foldout state for the noise properties |
| VariationProperties | Foldout state for the variation properties |

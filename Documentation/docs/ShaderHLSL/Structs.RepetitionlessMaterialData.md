# Structs/RepetitionlessMaterialData

## Description

Contains the data for a repetitionless material

## Variables

| Variable               | Description                                                                                                                                                                                                                                                                                                                                                                                          |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Settings               | Contains the compressed settings for the material<br>x (Setting Toggles):<br>> 0: Noise enabled<br>> 1: Randomise scaling<br>> 2: Randomise rotation<br>> 3: Smoothness enabled<br>> 4: Variation enabled<br>> 5: Packed texture enabled<br>> 6: Emission enabled<br>y (Assigned Textures):<br>> 0: Metallic<br>> 1: Smoothness<br>> 2: Roughness<br>> 3: Normal<br>> 4: Occlussion<br>> 5: Emission |
| TilingOffset           | Tiling offset for the material<br>xy: Tiling<br>zw: Offset                                                                                                                                                                                                                                                                                                                                           |
| AlbedoTint             | Tint for the albedo colour                                                                                                                                                                                                                                                                                                                                                                           |
| EmissionColor          | Emission colour if enabled                                                                                                                                                                                                                                                                                                                                                                           |
| Properties1            | Has some material properties<br>x: Metallic (0-1)<br>y: Smoothness (0-1)<br>z: Roughness (0-1)<br>w: Normal scale                                                                                                                                                                                                                                                                                    |
| Properties2            | Has some material properties<br>x: Occlussion strength (0-1)<br>y: Alpha clipping                                                                                                                                                                                                                                                                                                                    |
| NoiseSettings          | The base noise settings<br>x: Noise angle offset<br>y: Noise scale                                                                                                                                                                                                                                                                                                                                   |
| NoiseMinMax            | The noise random settings<br>xy: Random scaling min max<br>zw: Random rotation min max                                                                                                                                                                                                                                                                                                               |
| VariationMode          | The variation texture mode index<br>0: Perlin noise<br>1: Simplex noise<br>2: Custom texture                                                                                                                                                                                                                                                                                                         |
| VariationSettings      | The variation settings<br>x: Opacity<br>y: Small scale<br>z: Medium scale<br>w: Large scale                                                                                                                                                                                                                                                                                                          |
| VariationBrightness    | The brightness of the variation colour                                                                                                                                                                                                                                                                                                                                                               |
| VariationNoiseSettings | The variation noise settings<br>*Used if VariationMode is set to any noise*<br>x: Noise strength<br>y: Noise scale<br>zw: Noise offset                                                                                                                                                                                                                                                               |
| VariationTexture       | The variation texture<br>*Used if VariationMode is set to Custom Texture*                                                                                                                                                                                                                                                                                                                            |
| VariationTextureTO     | The variation texture tiling and offset<br>*Used if VariationMode is set to Custom Texture*<br>xy: Texture tiling<br>zw: Texture offset                                                                                                                                                                                                                                                              |

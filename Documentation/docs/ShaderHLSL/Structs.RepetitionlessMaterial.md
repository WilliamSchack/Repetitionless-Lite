# Structs/RepetitionlessMaterial

## Description

Contains the textures and data for a repetitionless material

Textures are only used if their respective settings are enabled

## Variables

| Variable      | Description        |
| ------------- | ------------------ |
| Albedo        | Albedo texture     |
| MetallicMap   | Metallic texture   |
| SmoothnessMap | Smoothness texture |
| RoughnessMap  | Roughness texture  |
| NormalMap     | Normal texture     |
| OcclussionMap | Occlussion texture |
| EmissionMap   | Emission texture   |
| Data          | The material data  |
